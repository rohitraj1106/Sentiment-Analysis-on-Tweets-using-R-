# Sentiment Analysis on Tweets using R

This project performs sentiment analysis on tweets using the R programming language. It leverages the `rtweet` package to fetch tweets, `tidytext` for text processing, and `dplyr` and `ggplot2` for data manipulation and visualization. [cite: 6, 7, 8]

## Introduction

Sentiment analysis is a Natural Language Processing (NLP) technique used to determine the emotional tone behind a piece of text. In this project, we analyze the sentiment of tweets related to a specific keyword. [cite: 11, 12, 13, 6, 7]
